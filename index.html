Installation

Install using the SDBG Plugin Update Site (http://sdbg.github.io/p2).

Beta Version

Install using the SDBG Plugin Beta Update Site (http://sdbg.github.io/p2beta).

Community

Any comments, bugs or issues should be directed to the SDBG Issues list at GitHub.

Status

SDBG is roughly on-par with the Chrome DevTools JavaScript debugger.

The Beta Version now supports Chromium, Edge, Chrome and Firefox

Some limitations:
  • (Issue #6) Variables are displayed in JavaScript (shared with Chrome DevTools).
  • (Issue #52) Support conditional breakpoints.
For more information check the SDBG Issues list and the README.md file.
